#!/bin/bash

# Update the system
echo "Updating the system..."
sudo apt update -y && sudo apt upgrade -y

# Install essential packages
echo "Installing essential packages..."
sudo apt install -y git-core zsh curl python3 python3-pip python3-venv libxml2-dev libglib2.0 build-essential libgtk-3-dev libgtksourceview-4-dev libpeas-dev libjalali-dev libxapp-dev android-tools-adb android-tools-fastboot

# Configure Git
echo "Configuring Git..."
git config --global core.editor /usr/bin/nano
git config --global user.email "mezaquegit@gmail.com"
git config --global user.name "Mezaque Silver"

# Install Oh My Zsh
echo "Installing Oh My Zsh..."
sh -c "$(wget -O- https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# Install packages required for compiling ROMs
echo "Installing packages for compiling ROMs..."
git clone --depth 1 git@github.com:akhilnarang/scripts scripts && cd scripts && setup/android_build_env.sh

# Install Python dependencies
echo "Installing Python dependencies..."
sudo apt-get install -y 2to3 dh-python python-is-python3 python3 python3.10 python3-pip apt python3-pylint-common

# Create Python virtual environment
echo "Creating Python virtual environment..."
python3 -m venv ~/android_build_env
source ~/android_build_env/bin/activate

# Finalization
echo "Configuration finished on Ubuntu!"

Automating the Process:
These scripts automate the setup of the entire build environment on Arch Linux and Ubuntu. They take care of:

System update.

Installation of essential packages and build tools.

Git, Zsh, and ADB configuration.

Installation of specific packages for compiling Android ROMs.

Python and dependency configuration.

With these scripts, you can quickly set up your environment by simply running a few commands in the terminal, saving time and effort in manual configuration.
